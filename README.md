# Isla Meyn - Elite Athlete Profile

A professional digital portfolio showcasing the athletic achievements, academic excellence, and competitive schedule of Isla Meyn.

**Live Site:** [islameyn.com](https://islameyn.com)

## Project Structure

```
isla_athlete/
├── index.html                 # Landing page
├── css/
│   └── styles.css            # Design system and custom styles
├── js/
│   └── main.js               # Core JavaScript functionality
├── images/                   # Athlete photos and media
├── protected/
│   ├── profile.html          # Performance records
│   ├── achievements.html     # Academic achievements
│   └── schedule.html         # Competition schedule
├── CNAME                     # GitHub Pages custom domain
├── package.json              # Project metadata
└── README.md                 # This file
```

## Features

- **Responsive Design** - Mobile-first approach using Tailwind CSS
- **Design System** - Material Design 3 color palette and typography
- **Performance Focused** - Lightweight HTML/CSS with minimal dependencies
- **SEO Optimized** - Proper semantic HTML and meta tags
- **Accessible** - WCAG compliant markup and navigation

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/thmeyn/isla_athlete.git
cd isla_athlete
```

2. Open in your browser:
- Local: Open `index.html` directly in your browser
- Live: Visit [islameyn.com](https://islameyn.com)

## Design System

The project uses a custom Material Design 3 theme with the following primary colors:
- **Primary**: #00020a (Deep Navy)
- **Secondary**: #bb0027 (Crimson Red)
- **Tertiary**: #000208 (Almost Black)

## Development

### Adding Content
- Update athlete information in `index.html`
- Add/modify images in the `images/` directory
- Update performance records in `protected/profile.html`
- Add academic achievements in `protected/achievements.html`
- Manage schedule in `protected/schedule.html`

### Styling
- Custom CSS is in `css/styles.css`
- Tailwind CSS utilities are available via CDN
- All design tokens are defined as CSS custom properties

### JavaScript
- Core functionality in `js/main.js`
- Handles navigation highlighting and smooth scrolling
- Extensible for additional interactivity

## Deployment

This site is hosted on GitHub Pages and uses a custom domain via CNAME.

### To Deploy:
1. Push changes to the `main` branch
2. GitHub Actions will automatically deploy to GitHub Pages
3. Changes will be live at islameyn.com within minutes
